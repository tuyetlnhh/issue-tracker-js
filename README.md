# Issue Tracker
### Link UI: (https://issues-tracker-js-linh.netlify.app/)

### Technical: 
  - HTML, CSS, Javascript
  - Bootstrap v5

### Feature:
  - Add todo
  - Search todo
  - Delete todo
  - Close todo (Change status from 'Open' to 'Closed')
  - Filter todo (Filter by status)
  - Order by todo (Sort todo by severity from low to high or high to low)
